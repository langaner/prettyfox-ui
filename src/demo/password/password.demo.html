  • fox-password - base class
  • fox-password__strong - strong container class
  • fox-password__strong-item - strong item class
  • fox-password__generate - generate button
  • name (string) - password name
  • placeholder (string) - placeholder
  • size (string) - password size: sm ,md,lg
  • disabled (boolean) - disabled status for password
  • buttonColor (string) - button color. Used bootstrap color classes: success ,info, warning, danger, primary, secondary
  • length (number) - generated password length
  • minLength (number) - min length of string to detect on of strength level

Langs

  • generate (string) - generate button text
  • generated - generat event. Returned {{ '{' }}password: password, strong: this.strongValue{{ '}' }}
  • clicked - click event on password. Returned {{ '{' }}originalEvent: event{{ '}' }}
  • changed - change event on password. Returned {{ '{' }}originalEvent: event, value: this.innerValue{{ '}' }}
