  • fox-pagination - base class
  • fox-pagination__item - page item
  • fox-pagination__link - page link
  • fox-pagination__prev - to prev page button
  • fox-pagination__next - to next page button
  • fox-pagination__first - to first page button
  • fox-pagination__last - to last page button
  • prevNext (boolean) - display next, prev buttons
  • lastFirst (boolean) - display last, first buttons
  • max (number) - max pages to display. Zero value is displayed all pages
  • separator (string) - separator string
  • total (number) - total pages
  • size (string) - pagination size: sm ,md,lg
  • color (string) - pagination color. Used bootstrap color classes: success ,info, warning, danger, primary

Langs

  • first (string) - first button text
  • prev (string) - prev button text
  • next (string) - next button text
  • last (string) - last button text
  • selected - click event on pagination. Returned {{ '{' }}originalEvent: event, page: page{{ '}' }}
  • nextClicked - next button click event on pagination. Returned {{ '{' }}originalEvent: event, page: page{{ '}' }}
  • prevClicked - prev button click event on pagination. Returned {{ '{' }}originalEvent: event, page: page{{ '}' }}
  • firstClicked - first button click event on pagination. Returned {{ '{' }}originalEvent: event{{ '}' }}
  • lastClicked - last button click event on pagination. Returned {{ '{' }}originalEvent: event{{ '}' }}
