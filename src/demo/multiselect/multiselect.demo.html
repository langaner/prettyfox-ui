  • fox-multiselect - base class
  • fox-multiselect__search - search form
  • fox-multiselect__wrapper - wrapper
  • fox-multiselect__divider - divider
  • fox-multiselect__actions - actionbar
  • fox-multiselect__dropdown - dropdown with options
  • fox-multiselect__item - option
  • color (string) - multiselect color. Used bootstrap color classes: success ,info, warning, danger, primary, secondary
  • size (string) - multiselect button size: sm ,md,lg
  • search (boolean) - view search form
  • closeOnSelect (boolean) - close on select
  • checkAll (boolean) - view check button
  • uncheckAll (boolean) - view uncheck button
  • titleMaxItems (number) - max items in titile
  • maxHeight (string) - max options dropdown height
  • single (boolean) - multiselect name
  • disabled (boolean) - disabled status for multiselect
