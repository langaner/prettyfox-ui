  • color (string) - tags color. Used bootstrap color classes: success ,info, warning, danger, primary, secondary
  • size (string) - tags size: sm ,md,lg
  • duplicates (boolean) - allow tag duplicates
  • addTagOnComma (boolean) - add tag on coma enter
  • addTagOnEnter (boolean) - add tag on enter push
  • addTagOnPaste (boolean) - add tag on past value from buffer
  • addTagOnSpace (boolean) - add tag on space push
  • addTagOnBlur (boolean) - add tag on blur
  • minTagLength (number) - min entered tag length
  • disabled (boolean) - disabled status for tags
  • autocomplete (boolean) - use autocomplete
  • autocompleteItems (array) - autocomplete items array
  • add - add tag. Returned tags: array
  • removed - remove tag. Returned tag: TagItem
