  • fox-button - base class
  • fox-switcher__container - switcher container

  • fox-switcher__true - true button
  • fox-switcher__false - false button
  • fox-switcher__label - label(center) button
  • trueColor (string) - color of true value: success ,info, warning, danger, primary, secondary
  • falseColor (string) - color of false value: success ,info, warning, danger, primary, secondary
  • labelColor (string) - center button color: success ,info, warning, danger, primary, secondary
  • trueValue (string) - true value
  • falseValue (string) - false value
  • size (string) - switcher size: sm ,md,lg
  • disabled (boolean) - disabled status for switcher
  • width (number) - switcher button width
