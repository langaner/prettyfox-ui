Prettyfox UI Components(alpha version)

Collection of native open source UI components for Angular2 styling by Bootstrap CSS v4. You don't need to include Bootstrap JS, jQuery or any other jQuery plugins.

Dependencies
  • Font awesome for nice icons https://www.npmjs.com/package/font-awesome
  • Moment for date manipulations https://www.npmjs.com/package/moment
  • PrismJS for code highlight https://www.npmjs.com/package/prismjs
  • Tinycolor for color manipulations https://www.npmjs.com/package/tinycolor2
